Purpose Of The AngularBootsrap-UI.

This User-Interface showcases not only showcases how to code in Angular Components, Angular Routing but also how a Developer can incorporate Bootstrap along with some custome styling. Its easier to refer to Bootstrap documentation to correct when mistakes arise when working in team. It is also easier to maintain and update Websites/Apps when they coded in components and that is using frame works such as REACT and Angular makes life of a Developer much easier. My Navbar, HomePage, AboutPage,ContactPage, and Footer were all coded in seperate components making it easier to make changes.
